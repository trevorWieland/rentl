# Standards: Documentation Overhaul for v0.1 Release

## Applicable Standards

1. **ux/copy-pasteable-examples** — All command examples in docs must be executable without modification
2. **ux/stale-reference-prevention** — Cross-reference audits must verify against actual CLI output and config files
3. **ux/frictionless-by-default** — Guided setup and safe defaults for effortless first runs
4. **global/no-placeholder-artifacts** — Never commit placeholder values; all artifacts must be functional and verified
